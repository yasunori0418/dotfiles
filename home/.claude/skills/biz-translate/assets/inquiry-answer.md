# ビジネス職からの質問への回答テンプレート

ビジネス職からエスカレされた技術的な質問に答える用。
「ビジネス職が理解するための回答」と「ビジネス職が顧客にそのまま返せる文面」を分けて出すのがポイント。

```
## ご質問への回答

### 結論
[質問に対する答えを一言で。Yes/No が問われているなら明確に]

### 理由・背景
[なぜそうなるかを、技術メカニズムではなく業務・仕様の言葉で]

### お客様への回答例（そのまま転送可）
> [ビジネス職が顧客に返せる、です・ます調の文面。技術用語ゼロ]

### 補足（社内向け・社外には出さない）
[ビジネス職が把握しておくと良い前提や、聞かれたら答えられる範囲のメモ]
```

---
### 送付前チェック（エンジニア確認用・社外には出さない）
- [ ] 「結論」は質問に正面から答えているか
- [ ] 「お客様への回答例」に技術用語・内部名が残っていないか
- [ ] 確実でないことを断定していないか
- [ ] 社内向け補足が顧客向け文面に混ざっていないか
